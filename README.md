# web-builder-react
# (Important) Change in  
```powershell
npx create-react-app 
```
# after run the command above, create a file in the app root directory
```powershell
nano .env
```
write this line shown below into the file
```linux
SKIP_PREFLIGHT_CHECK=true
```
then you can start it by command below
```powershell
npm start
```
